# agent-workbench

MCP servers that give AI agents real development tools - symbol-aware editing, persistent processes, git history, type checking, and more.

## Quick Start

```bash
npm install && npm run build
```

Then configure your MCP client. See [Getting Started](docs/getting-started.md).
